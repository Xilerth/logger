It's just an Angular LOGGER

Installation

Install the package via npm:

npm install @Xilerth/ngx-logger

Configuration

Import and inject the service in your component.

import { Component, inject } from '@angular/core';
import { LoggerService } from '@xilerth/ngx-logger'

@Component({
  selector: 'my-component',
  templateUrl: './my-component.component.html',
  styleUrls: ['./my-component.component.css']
})

export class MyComponent {

  private loggerService: LoggerService = inject(LoggerService);

}

Use

Use the service to log messages.

this.loggerService.log({
  title: '[Info Message]',
  message: 'Hello World!',
  type: 'info',
  color: 'blue'
  weight: 'bold'
  showTime: true,
  disabled: false
})

Logger Interface

Logger {
  title: string; (Optional: null)
  message: string; (Required)
  type: LoggerType; (Required)
  color: string;  (Optional: null)
  weight: string; (Optional: null)
  showTime: boolean; (Optional: false)
  disabled: boolean; (Optional: false)
}

Extra Functionalities

Disable logs per Logger Type

this.loggerService.disableLog('info');

Save logs and subscribe to them thanks to the observable!

this.loggerService.logRecords$().subscribe(logs => {
  ...
})
